## Compiling a C Program
gcc is a GNU c compiler
´´´
// -o is that it is an output if not it will create an a.out file
$ gcc -o hello hello.c
// . means current directory
$ ./hello
I am Sam
´´´

## Compiling a C++ Program
g++ is a GNU c++ compiler
´´´
$ g++ -o hello hello.cpp
$ ./hello
I am Sam
´´´
